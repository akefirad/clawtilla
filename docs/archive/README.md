# Archived design records

These are the historical design documents preserved for their rationale and
`oss/clawpatrol` source citations. They captured the design as it was worked out
(discovery → plan) and are **superseded** as the description of the current system
by [`../architecture.md`](../architecture.md).

Read them when you need the *why* behind a decision or the exact upstream
citations; read `../architecture.md` for what the system is today.

- [discovery.md](./discovery.md) — problem framing, the design space, decisions,
  open questions, and source citations.
- [plan.md](./plan.md) — the build sequence, concrete artifacts, and the
  "Known risks" log (including the items later resolved at runtime).
